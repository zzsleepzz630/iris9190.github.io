# 2018-10-02-Making-a-technical-blog.md

## 블로그 만드는 법 

	- 원하는 블로그의 Github 계정을 알아본다 
	- 해당 github repository에서 fork 한다 
	- 레파지토리의 이름을 account.github.io로 바꾼다 
	- 자신이 원하는 설정으로 바꾼다. 
	- 콘텐츠를 업데이트 한다.
